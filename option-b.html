What the NOC Enables

Centralized Network Management

Single UniFi controller for all 6 sites. Network status, device health, configuration from one location.

Proactive Monitoring

Alerts before problems impact service. Catch network issues, system failures, connectivity problems early.

Remote Troubleshooting

Fix issues remotely without physical site visits. Reboot equipment, diagnose problems, restore service from Whitney.

Network Independence

Local UniFi controller (not cloud-dependent). Complete network control, faster response, data sovereignty.

Future-Ready Platform

Infrastructure ready for Site 8, 9, 10+. Portal, AI tools, API integrations can be added when ready (see Option C).

Professional Documentation

Asset mapping, network topology, configuration records. Any competent technician can work with this foundation.

Pathway to Portal & AI

Option B builds the infrastructure foundation. The NOC computer, network connectivity, and centralized management create the pathway for future development.

What's NOT delivered in Option B: Portal, API integrations, AI assistants.

What IS delivered: The infrastructure that makes those things possible later.

If you want Portal and AI delivered now, see Option C — but note it requires partnership with AYWD and uses a different CCTV ecosystem (UniFi Protect instead of Hikvision).
